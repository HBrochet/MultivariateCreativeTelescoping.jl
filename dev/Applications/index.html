This section describes concrete problems that have been solved using this package.

Counting k-regular graphs

A graph is $k$ regular if all its vertices have degree exactly $k$. Let $u(n,k)$ be the number of $k$-regular graphs on $n$ vertices. We are interested in finding a recurrence relation on $u(n,k)$ for fixed values of $k$. Chyzak and Mishna were able to compute such a recurrence for $k$ up to 7 [link]. The computation for k=7 took 24 weeks of CPU time. With this package, we were able to obtain the same result in 7 minutes and to compute the recurrence for k=8.

Example

Below is the script for k=6. It returns a linear differential equation annihilating the generating series of the $u(n,6)$. Excluding compilation time this computation takes approximately 3 seconds and returns an operator of order 6 and degree 213.
